# 🎓 Student Management System in C

A console-based Student Management System developed in **C** using **Structures**, **Functions**, and **File Handling**. This project allows users to manage student records efficiently through a menu-driven interface.

---

## ✨ Features

- ➕ Add Student Record
- 📋 Display All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- ❌ Delete Student Record
- 🏫 Display Students Department-wise
- 📊 Calculate Department-wise Average Marks
